# La-paradox-du-Master-Felix
Le problème du pirate.

Un pirate arrive sur l'île du trésor. Il trouve N coffres. Un seul coffre contient le trésor. Les autres donne un indice. L'indice donne un chiffre entre 0 et N, c'est la position où le coffre ne s'y trouve pas. Ainsi qu'un signe + ou -. + signifie que le trésor se trouve à gauche de cette position. et - le contraire, le trésor se trouve à droite.

Code treasures.py: retourne un array de N éléments selon la paradox du Master Felix.
